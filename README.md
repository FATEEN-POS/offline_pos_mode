# 🚀 نظام "فطين" الذكي لنقاط البيع (Fatin Smart POS)
### نظام مبيعات ومحاسبة محلي (Offline-First) مخصص للسوق المصري (أسوان).

نظام متكامل مصمم للعمل بكفاءة عالية في المحلات التجارية (سوبر ماركت، محلات ملابس، مطاعم) مع التركيز على السرعة، الاستقرار بدون إنترنت، والتوافق مع قوانين مصلحة الضرائب المصرية.

## 🛠 التقنيات المستخدمة (Tech Stack)
* **الواجهة الأمامية (Frontend):** HTML5 & CSS3 (نظام متجاوب Responsive).
* **المنطق البرمجي (Logic):** Vanilla JavaScript (لضمان السرعة والعمل بدون خادم).
* **التخزين المحلي (Storage):** IndexedDB / LocalStorage (تخزين بيانات المنتجات والفواتير محلياً).
* **التكامل (Integrations):** دعم قارئ الباركود (HID) وطابعات الإيصالات الحرارية (Bluetooth/USB).

## ✨ الميزات الرئيسية
* **Offline-First:** يعمل النظام بالكامل بدون الحاجة لاتصال بالإنترنت، مع إمكانية المزامنة لاحقاً.
* **Barcode Scanning:** دعم المسح الضوئي للأصناف باستخدام الكاميرا أو القارئ اليدوي.
* **Egyptian Tax Ready:** حساب تلقائي لضريبة القيمة المضافة (14%) وتنسيق الفاتورة الضريبية.
* **Bluetooth Printing:** موديول جاهز لإرسال الأوامر للطابعات الحرارية (Thermal Receipt Printers).
* **Smart Analytics:** تقارير مبيعات فورية وتحليل للمخزون (Stock Management).

## 📝 متطلبات التشغيل في مصر
1.  **المعدات:** جهاز لوحي (Tablet) أو كمبيوتر بمخرج USB لربط القارئ.
2.  **الطباعة:** طابعة حرارية 58mm أو 80mm.
3.  **الضرائب:** يجب إدخال رقم التسجيل الضريبي في الإعدادات لتظهر في الفاتورة القانونية.

قم بفتح ملف `index.html` في أي متصفح حديث (Chrome/Edge)، وسيعمل النظام مباشرة دون الحاجة لضبط سيرفر خارجي.
